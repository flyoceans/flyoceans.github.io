Quick Start

1 Hexo

Hexo 就是一个博客框架，用node.js 实现的，轻量，快速。可以支持你用Markdown写blog，即使你不会前端(I personally hate node.js?)。
部署可以在github或 heroku。用它省去了很多造轮子的问题，各种插件也很完善了，当然高阶的话。。。
